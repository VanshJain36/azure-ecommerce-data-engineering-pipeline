# Azure E-Commerce Data Engineering Pipeline

End-to-end Azure Data Engineering project built using:

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- PySpark
- Delta Lake
- Power BI

## Architecture

Source CSV
→ ADF
→ Bronze Layer
→ Silver Layer
→ Gold Layer
→ KPI Layer

## Features

- Medallion Architecture
- Data Quality Checks
- Star Schema
- Revenue Trends KPI
- Top Products KPI
- Customer Retention KPI
- Delivery Delay KPI
- Order Status KPI
